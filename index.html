Privacy Policy

Effective date: 26 April 2026

SparkDocs is an electrical certification application used to create and manage electrical installation certificates and related documentation.

Information We Collect

  • Certification data entered by the user
  • Photos captured using the device camera (if used)
  • Basic device information for compatibility purposes

How We Use Data

  • Generate electrical certificates (EIC, EICR, MWC, etc.)
  • Attach photographic evidence to job records
  • Export PDF documentation

Camera Usage

The camera is only used when the user chooses to capture images for documentation purposes. Images remain under user control.

Data Storage

Data may be stored locally on the device or in secure cloud storage if enabled. We do not sell or share personal data with third parties.

Contact

If you have questions, contact the developer via email.
